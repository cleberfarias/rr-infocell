# Backend

Pasta reservada para a API do MVP do RR Infocell.

## Stack definida

- Node.js
- Express
- TypeScript
- Firebase Admin SDK
- Firebase Auth
- Firestore

## Responsabilidades

- Validar regras de negocio.
- Proteger operacoes privadas.
- Gerar numero sequencial de ordem de servico.
- Centralizar escrita em colecoes sensiveis.
- Registrar historico de alteracoes.
- Expor endpoints para o frontend.

## Modulos previstos

- `auth`
- `usuarios`
- `clientes`
- `aparelhos`
- `ordens-servico`
- `checklists`
- `estoque`
- `relatorios`
