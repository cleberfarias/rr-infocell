# RR Infocell

Sistema web para gestão de assistência técnica e loja de celulares.

## Stack atual

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Firebase como base definida para o MVP

## Scripts

```bash
npm install
npm run dev
npm run build
npm test
```

## Design system

A base visual do projeto está documentada em [`docs/design-system.md`](docs/design-system.md).

Componentes de produto ficam em `src/components/design-system`.
